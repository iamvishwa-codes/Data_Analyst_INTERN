# Task 1: Data Cleaning and Preprocessing

## Objective
Clean and preprocess a raw dataset by handling missing values, removing duplicates, standardizing formats, and fixing data types.

## Dataset
Netflix Movies and TV Shows Dataset (Kaggle)

## Tools Used
- Python
- Pandas
- VS Code

## Data Cleaning Steps
- Loaded raw dataset using Pandas
- Handled missing values using fill and drop methods
- Removed duplicate records
- Cleaned column names (lowercase, snake_case)
- Standardized text values
- Converted date columns to datetime format
- Fixed incorrect data types
- Exported cleaned dataset

## Output
- cleaned_netflix_data.csv

- Data Cleaning
- Data Preprocessing
