Project Overview
This project focuses on analyzing a cleaned e-commerce dataset using SQL.
The goal was to import data from CSV files, clean it, perform analysis, and optimize queries using proper SQL techniques.

It was exciting to work with real-world data and understand how databases are used in practical business scenarios.

Objective
- Import structured data into a SQL database
- Clean and prepare data for analysis
- Perform analytical queries using SQL
- Apply joins, subqueries, and aggregations
- Create views for reusable analysis
- Optimize queries using indexes

Tools Used
- MySQL
- VS Code
- CSV Files
- Olist E-commerce Dataset

Dataset
The Olist e-commerce dataset was used for this project.
Only required columns were selected and cleaned before import to avoid errors and reduce complexity.
This made the dataset lightweight and easy to work with.

Project Structure

Steps Performed
1. Data Import  
   Created database and tables, then imported cleaned CSV files into MySQL.

2. Data Cleaning

3. Data Analysis  
   Used SELECT, WHERE, ORDER BY, GROUP BY, joins, subqueries, and aggregate functions to analyze the data.

4. Views and Optimization  
   Created SQL views for summarized insights and added indexes to improve query performance.
